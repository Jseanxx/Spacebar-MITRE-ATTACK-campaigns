2026 Seoul Dental BookingPress Intrusion

2026 Seoul Dental BookingPress Intrusion은 치과 사이트인 Seoul Dental Clinic의 WordPress 기반 웹 서버와 환자 데이터베이스를 대상으로 한 침해 캠페인이다. Gsin1626은 대상 사이트의 Contact 페이지에 공개된 관리자 이메일 주소를 사전에 확보하여 BookingPress 플러그인의 인증 우회 취약점(CVE-2024-7350)을 이용해 관리자 계정으로 WordPress에 로그인하였다. 이후 테마 편집기를 통해 functions.php에 PHP 웹쉘을 삽입하여 원격 코드 실행 경로를 확보하였다. Gsin1626은 cron 설정 파일이 취약한 권한(777)의 파일을 root 권한으로 실행하고 있다는 점을 이용하여 SSH 공개키를 주입함으로써 root 계정 원격 접속 권한을 획득하였다. 최종적으로 MySQL 데이터베이스에 접근하여 환자 진료 정보가 담긴 patient_charts 테이블을 덤프하고 SCP를 통해 자신의 PC로 유출하였다.
