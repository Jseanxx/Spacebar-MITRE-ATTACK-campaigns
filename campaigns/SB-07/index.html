OZZY PMS Chain

OZZY PMS Chain은 AD 환경과 JBoss/PMS 업데이트 경로 간의 신뢰 관계를 악용하는 침해 캠페인이다. 공격자는 bastion을 통해 내부망에 진입한 뒤 hanguel-ops-pms의 JBoss AS 6 취약 표면을 이용해 PMS 업데이트 파일과 manifest를 변조한다. hanguel-win01의 Hanguel PMS Agent는 변조된 패치를 nt authority\system 권한으로 실행하고, 이 과정에서 DC 접근에 필요한 credential artifact가 노출된다. 이후 공격자는 HANGUEL\Administrator 권한으로 hanguel-dc01의 관리자 공유(C$)와 WinRM 원격 관리 채널에 접근 가능한 상태가 되며, PMS 업데이트 체인의 신뢰 훼손은 도메인 컨트롤러 credential access 위험으로 확대된다.

Overview

이 캠페인은 외부 접근 경로, Linux 기반 JBoss/PMS 서버, Windows PMS Agent, Active Directory 도메인 컨트롤러를 기업형 운영망 환경으로 구성하여 침해 흐름을 보여주는 캠페인이다. 핵심 전개는 hanguel-ops-pms의 업데이트 경로의 패치파일을 변조해 hanguel-win01에서 SYSTEM 실행 흐름을 만들고, hanguel-win01에서 발견한 credential artifact를 복호화 가능한 컨텍스트로 처리하여 HANGUEL\Administrator 접근 정보를 확보하는 것이다. 이후 공격자는 DC의 관리자 공유(C$) 및 WinRM 원격 실행 채널을 이용해 로더와 암호화된 payload를 배치하고, 로더가 payload를 메모리에서 복호화 및 manual mapping하여 LSASS dump를 생성하는 흐름을 구성한다.

Techniques Used

Phase ID Name Use
Initial Foothold T1133 External Remote Services 공격자는 제공된 SSH 접근 경로를 이용해 hanguel-bastion에 접속하고, 이를 내부망 10.60.0.0/16으로 진입하기 위한 거점으로 사용한다.
Discovery T1046 Network Service Discovery 공격자는 hanguel-bastion에서 내부망을 스캔하여 hanguel-ops-pms:8080, hanguel-dc01:{445,3389,5985}, hanguel-win01:{3389,5985} 등 주요 서비스를 식별한다.
Initial Access T1190 Exploit Public-Facing Application 공격자는 hanguel-ops-pms의 JBoss AS 6 취약 표면을 이용해 PMS 서버 컨테이너 내부에서 명령 실행 권한을 확보한다.
Execution T1059.004 Unix Shell 공격자는 JBoss 컨테이너 shell에서 id, hostname, pwd, 네트워크 경로, 배포 디렉터리를 확인하여 실행 위치와 권한을 파악한다.
Persistence T1505.003 Web Shell 공격자는 JBoss 배포 경로에 JSP webshell 또는 marker를 배치하여 PMS 서버에서 지속적인 명령 실행 채널을 유지한다.
Discovery T1083 File and Directory Discovery 공격자는 PMS 서버 내부에서 JBoss 배포 경로와 Windows PMS Agent가 신뢰하는 /updates/manifest.json, /updates/patch.ps1 업데이트 구조를 확인한다.
Supply Chain T1195.002 Supply Chain Compromise 공격자는 PMS 업데이트 경로의 patch.ps1과 manifest를 변조하여 내부 Windows 클라이언트가 신뢰하는 정상 패치 배포 흐름에 공격 코드를 삽입한다.
Defense Evasion T1036.005 Masquerading 공격자는 변조 패치를 기존 파일명과 경로인 /updates/patch.ps1로 유지하고 manifest 버전/해시만 갱신하여 정상 업데이트처럼 보이게 만든다.
Execution T1053.005 Scheduled Task / Job hanguel-win01의 Hanguel PMS Agent Scheduled Task가 PMS 서버에서 변조 패치를 다운로드하고 nt authority\system 권한으로 실행한다.
Discovery T1082 System Information Discovery 공격자는 win01 SYSTEM 컨텍스트에서 사용자, 호스트명, IP 구성, DNS 서버, 실행 권한을 확인하여 감염된 클라이언트의 내부 위치와 권한 수준을 파악한다.
Discovery T1482 Domain Trust Discovery 공격자는 win01에서 hanguel.local 도메인과 도메인 컨트롤러 hanguel-dc01.hanguel.local / 10.60.20.10 존재를 확인한다.
Discovery T1018 Remote System Discovery 공격자는 hanguel-dc01의 SMB, WinRM, LDAP, Kerberos, DNS 서비스 접근 가능성을 확인하고 후속 장악 대상으로 지정한다.
Credential Access T1552 Unsecured Credentials 공격자는 win01의 C:\ProgramData\HanguelPMS\dc_cred.xml credential artifact를 발견하고, 복호화 가능한 사용자 컨텍스트에서 HANGUEL\Administrator 자격 증명을 확보한다.
Lateral Movement T1021.002 SMB / Windows Admin Shares 공격자는 HANGUEL\Administrator 자격 증명을 이용해 win01에서 DC의 \\10.60.20.10\C$ 관리 공유에 접근하고, hgl_loader.exe, hgl_payload.enc, hgl_run.cmd를 C:\Windows\Temp에 배치한다.
Lateral Movement T1021.006 Windows Remote Management 공격자는 HANGUEL\Administrator 자격 증명을 이용해 win01에서 DC에 WinRM 명령을 실행하고, DC 내부에서 SYSTEM 권한 Scheduled Task를 등록한다.
Defense Evasion T1027 Obfuscated Files or Information 공격자는 LSASS dump payload DLL을 XOR 암호화된 hgl_payload.enc 형태로 전달하여 정적 파일 탐지 가능성을 줄인다.
Defense Evasion T1620 Reflective Code Loading 공격자는 hgl_loader.exe를 이용해 암호화된 PE/DLL payload를 메모리에서 복호화하고, PE header/section mapping, relocation, import resolving을 직접 수행하여 디스크에 복호화된 DLL을 남기지 않고 실행한다.
Credential Access T1003.001 OS Credential Dumping: LSASS 공격자는 DC의 SYSTEM Scheduled Task에서 manual-mapping loader를 실행해 lsass.exe 메모리를 C:\Windows\Temp\hgl_diag.bin으로 dump하고, 이를 win01로 회수한다.
Credential Access T1003.001 LSASS Credential Material 공격자는 win01에서 Mimikatz로 DC LSASS dump를 offline 분석하여 HANGUEL\Administrator NTLM hash와 Kerberos key material을 확보한다.
Lateral Movement T1550.002 Pass the Hash 공격자는 LSASS dump에서 확보한 HANGUEL\Administrator NTLM hash를 이용해 평문 비밀번호 없이 DC 및 다른 Windows 시스템에 SMB/WinRM 기반 관리자 접근을 시도할 수 있다.

S-SB-007-03 Windows Scheduled Task win01의 PMS Agent 실행과 DC SYSTEM 컨텍스트 작업 등록에 사용된 Windows 작업 스케줄러 기능이다. Execution mechanism
