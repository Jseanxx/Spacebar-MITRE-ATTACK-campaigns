KL-001 Kyverno Unsafe Pod Creation Log

Log ID:
KL-001
Source:
SB-04 K8s Node
Representative Path:
/var/log/nodefall/nodefall-events.log
Collection:
Wazuh agent file input (/var/log/nodefall/nodefall-events.log)
Primary Use:
특권 Pod 생성 및 호스트 네임스페이스 접근이 가능한 위험 Pod 스펙 생성 행위 확인

공격 행위 요약

KL-001은 NodeFall 캠페인에서 공격자가 Kubernetes API 권한을 악용해 컨테이너 탈출에 사용할 수 있는 위험한 Pod를 생성한 행위를 설명하는 Kyverno 기반 로그이다.

공격자는 확보한 Kubernetes 자격 증명과 권한을 이용해 privileged, hostPID, hostNetwork, hostIPC, hostPath와 같은 위험한 Pod 스펙을 포함한 Pod 생성을 시도했다. 이러한 설정은 컨테이너가 호스트의 네임스페이스나 파일 시스템에 접근할 수 있게 만들어 컨테이너 격리 경계를 약화시킨다.

Kyverno는 Kubernetes API admission 단계에서 Pod 생성 또는 수정 요청을 검사하며, 위험한 스펙이 포함된 요청이 정책 조건과 매칭되면 PolicyViolation 이벤트를 생성한다. 이 로그는 실제 런타임 탈출 행위가 발생하기 전, 공격자가 호스트 접근을 가능하게 하는 실행 환경을 준비했다는 증거로 사용된다.

구분위치설명
