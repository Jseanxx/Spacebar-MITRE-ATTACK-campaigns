# React2Shell WIC_Infiltrator

본 캠페인은 WIC(Water Infrastructure Center) 포럼 포털의 Next.js Server Actions에서 발생하는 취약점을 거점으로 합니다. 공격자는 전역 프로토타입 오염 기법을 통해 초기 침투에 성공한 후, Node.js 런타임의 http 모듈을 가로채는 방식으로 은밀한 웹쉘을 구축했습니다. 이후 Docker 환경의 설정 미비(Root 권한 실행)를 악용하여 컨테이너 제어권을 완전히 장악하였으며, 이를 통해 정수장 제어망 및 SCADA 시스템으로의 측면 이동(Lateral Movement)을 시도한 고도화된 공급망 침해 시나리오를 포함합니다.
