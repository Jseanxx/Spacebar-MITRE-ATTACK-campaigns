<h1>Grafana Dashboard Compromise</h1>

<p class="summary">
  Grafana Dashboard Compromise는 Grafana CVE-2025-4123 XSS를 이용해 로그인된 victim의
  Grafana 브라우저 컨텍스트에서 API 데이터를 수집하고, 수집 결과를 공격자 인프라로 전송한
  침해 캠페인이다. 수집 데이터는 localStorage에 임시 저장된 뒤 HTTP로 공격자 서버에 전송되며,
  공격자 인프라에서 압축 및 Mock Cloud Storage 업로드가 수행된다.
</p>
