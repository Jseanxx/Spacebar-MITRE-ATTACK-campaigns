# 2024 PipelineKey

2024 PipelineKey는 침해된 개발자 단말 또는 탈취된 개발자 VPN 세션을 통해 내부 개발망에 접근한 뒤, Jenkins CI/CD 서버와 Staging Web Server를 대상으로 전개된 침해 캠페인이다. 공격 그룹 BreadKimchi는 개발자 단말에서 Jenkins Web UI와 CLI 접근 가능성을 확인하고 Jenkins 서버를 식별했다. 이후 Jenkins CLI 파일 읽기 취약점과 배포 credential 관리 미흡을 이용해 Job 설정, build log, workspace, 배포 키 단서를 조사했다. BreadKimchi는 Jenkins가 배포하는 Cat Cafe Staging 서버에 접근하여 nginx 설정, 배포 스크립트, 고객 정보와 예약 데이터를 수집하고, 수집한 데이터를 archive로 묶어 SSH/SCP를 통해 공격자 서버로 전송했다.
