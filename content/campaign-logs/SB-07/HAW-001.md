# HAW-001 Win01 Hanguel AD Agent Enriched Event

| Field | Value |
| --- | --- |
| Log ID | HAW-001 |
| Source | `hanguel-win01` |
| Representative Path | `C:\ProgramData\HanguelADAgent\logs\*.jsonl` |
| Collection | Hanguel AD Agent PowerShell/Python collector -> SOC HTTP/Logstash -> `hanguel-ad-agent-*` |
| Current Status | 수집 체계 구현 및 ELK 이벤트 확인 |
| Primary Use | win01에서 발생한 PMS patch 후속 행위를 하나의 정규화 이벤트 흐름으로 추적 |

## 공격 행위 요약

PMS Agent가 변조된 patch를 SYSTEM으로 실행하면 win01에서 권한, 네트워크, DNS, 도메인, DC 서비스 접근성을 확인하는 이벤트가 발생한다. 이 로그는 원본 명령 결과를 요약하고 `event.action`과 ATT&CK technique을 붙여 SIEM에서 바로 pivot할 수 있게 한다.

## Analyst Hunting KQL

이 로그는 원본 Windows Event Log가 아니라 Hanguel AD Agent가 명령 결과를 요약한 enriched event다. 분석가는 먼저 명령어, DC IP, credential artifact 경로 같은 행위 단서를 기준으로 검색하고, 이후 `event.action`으로 좁힌다.

```text
host.name:"hanguel-win01" and message:(*whoami* or *ipconfig* or *nltest* or *Resolve-DnsName* or *Test-NetConnection*)
```

```text
host.name:"hanguel-win01" and message:(*"10.60.20.10"* or *"hanguel-dc01"* or *"hanguel.local"*)
```

```text
host.name:"hanguel-win01" and message:*"C:\\ProgramData\\HanguelPMS\\dc_cred.xml"*
```

```text
host.name:"hanguel-win01" and message:(*"nt authority\\system"* or *"TcpTestSucceeded : True"* or *"_ldap._tcp.dc._msdcs.hanguel.local"*)
```
