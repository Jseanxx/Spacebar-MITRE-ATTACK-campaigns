# HAW-001 Win01 Hanguel AD Agent Enriched Event

| Field | Value |
| --- | --- |
| Log ID | HAW-001 |
| Source | `hanguel-win01` |
| Representative Path | `C:\ProgramData\HanguelADAgent\logs\*.jsonl` |
| Collection | Hanguel AD Agent PowerShell/Python collector -> SOC HTTP/Logstash -> `hanguel-ad-agent-*` |
| Current Status | 수집 체계 구현 및 ELK 이벤트 확인 |
| Primary Use | win01에서 발생한 PMS patch 후속 행위를 하나의 정규화 이벤트 흐름으로 추적 |

## 공격 행위 요약

PMS Agent가 변조된 patch를 SYSTEM으로 실행하면 win01에서 권한, 네트워크, DNS, 도메인, DC 서비스 접근성을 확인하는 이벤트가 발생한다. 이 로그는 원본 명령 결과를 요약하고 `event.action`과 ATT&CK technique을 붙여 SIEM에서 바로 pivot할 수 있게 한다.
